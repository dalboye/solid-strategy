# solid-strategy

## How to run
`cd solid-strategy`
`python3 -m venv solidenv`  
`source solidenv/bin/activate`     
`pip install -r requirements.txt`  
`python ec.py`  

## How to stop
After running the application, press any key when the picture is selected.
